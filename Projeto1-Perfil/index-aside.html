Dados

O Jogo do Dinossauro (também conhecido como Chrome Dino ) é um jogo para navegador desenvolvido pelo Google e integrado ao navegador Google Chrome . No jogo, o jogador guia um Tiranossauro rex pixelado por uma paisagem desértica com rolagem lateral .

Histórico

Foi criado em 2014 pela equipe Chrome UX como um easter egg para entreter usuários sem internet.
