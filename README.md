## HTML para eu do futuro!

 HTML significa HyperText Markup Language e é a
linguagem padrão de marcação para estruturar páginas web, definindo elementos como títulos, parágrafos, imagens e links.
Organiza o conteúdo com tags (ex: <p> para parágrafos, <h1> para títulos).
Incorpora textos, imagens, vídeos, áudios e tabelas.
Conecta páginas entre si através de links.
Utiliza tags específicas (<header>, <article>) para dar significado ao conteúdo.
